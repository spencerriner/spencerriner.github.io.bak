about

My name is Spencer Riner. I am an aspiring programmer, web developer, and musician.

I am a Computer Information Systems major at University of Houston Clear Lake. While pursuing this degree I've gained knowledge about various aspects of computer science and the principles of programming, including object-oriented programming using Java, principles of data structures, and computer organization.

I created this website using Twitter Bootstrap 3 and Less. You can find the source code on my GitHub repository. What you are seeing now is my third prototype, produced after a lot of trial and error. It is mobile friendly, but looks best in a modern browser like Mozilla Firefox or Google Chrome. Along with web design, I'm in the process of learning jQuery alongside Javascript to implement in my next web project.

There are a lot of other things I like too. I play guitar, electric bass, and dabble in audio production on a laptop. You can find some of the music I've worked on at our Bandcamp and SoundCloud websites.
